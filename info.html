Developer

Created by ValTheOneAndOnly, a solo developer focused on simplifying Windows optimization. Nexus is a portable, single-executable tool with no installation required and no background services.

Overview

58 registry, service, and power tweaks spanning CPU, GPU, System, and Network categories — including MMCSS foreground priority and network throttling removal for lower latency. Each tweak toggles independently and applies instantly. A System Restore Point is created automatically before the first change. Live hardware monitoring with CPU, multi-GPU, RAM, and Disk sparkline graphs — GPU per-engine grouping matches Task Manager with NVML fallback for NVIDIA cards. All toggle states are saved to the registry and restored on next launch.
